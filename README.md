#  Spot the ISS 

Run this script overnight and keep an eye on your email notifications.  
If you're lucky, you might get to **see the International Space Station (ISS)** flying over your location.

This is a simple, API-based Python project perfect for beginners.

---

##  Required Changes Before Running the Code

### 1. **Update Your Coordinates**
Edit the latitude and longitude values in the script to match **your local location**.

### 2. **Update Email Addresses**
Modify both:
- **Sender email address**
- **Receiver email address**

### 3. **Update Your App Password**
Replace the placeholder with the **actual app password** of your sender email account.

### 4. **Adjust Timezone Conversion**
The provided `convert_ist()` function works for **IST (Indian Standard Time)**.  
If you're in another timezone, update this function accordingly.
